# KizunaSutra

# KizunaSutra — Japanese Learning Website for MEXT Preparation

## Project Overview

**KizunaSutra** is a browser-based Japanese learning system built to support MEXT scholarship preparation.

Meaning Breakdown
Kizuna (絆) = bond, connection, human ties
Sutra = thread, guiding structure, connected path

This project is not just a learning website.
It is a structured system that shows:

* Commitment to studying in Japan
* Daily self-learning discipline
* Web development ability
* Practical problem-solving

The website should feel simple, clean, functional, and easy to use.

---

# Goal of the Website

Create a Japanese learning platform that helps users:

* Learn Hiragana and Katakana
* Practice vocabulary
* Track progress
* Take daily quizzes
* Stay consistent with Japanese study

### 5. Practice Quiz Page

### Purpose

Allow daily practice.

### Quiz Types

* Hiragana recognition
* Katakana recognition
* Vocabulary meaning

### Quiz Flow

1. Show question
2. User selects answer
3. Score updates
4. Next question loads
5. Final score displayed

### Example

Question:

Options:

* a
* i
* u
* e
